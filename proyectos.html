Proyecto 1

Desarrollo de una página web aplicando los conocimientos de 2º DAW

Falla
Este proyecto consiste en el desarrollo de una página web para una falla, incorporando un sistema de gestión que facilita la administración de contenidos y recursos. Actualmente, el proyecto se encuentra en fase de desarrollo y está siendo realizado de forma colaborativa. En su creación se están aplicando todos los conocimientos adquiridos durante el segundo curso del grado superior de Desarrollo de Aplicaciones Web (DAW), utilizando la metodología ágil Scrum para la organización del trabajo, la planificación de tareas y la mejora continua del proyecto.
Proyecto 2

Desarrollo de una empresa ficticia aplicando todos los conocimientos adquiridos durante el grado medio de Sistemas Microinformáticos y Redes.

Flolabs
Este proyecto consistió en la integración de todos los conocimientos adquiridos durante el grado medio de Sistemas Microinformáticos y Redes (SMR). El objetivo principal fue aplicar de manera práctica lo aprendido a lo largo del ciclo formativo mediante la creación y desarrollo de una empresa ficticia. A través de este proyecto, se pusieron en práctica competencias técnicas y organizativas como la gestión de sistemas, redes, soporte técnico y planificación empresarial, permitiendo una visión global y realista del funcionamiento de una empresa del sector tecnológico.
Proyecto a distancia

Proyecto colaborativo realizado de forma remota con estudiantes de Miami, orientado al aprendizaje del inglés y al trabajo en equipo en entornos virtuales.

Miami
Este proyecto se desarrolló de manera remota en colaboración con un grupo de estudiantes de Miami. El trabajo se realizó a distancia y la comunicación se llevó a cabo en inglés, mientras que los contenidos producidos por el equipo de Miami fueron realizados en español. El objetivo principal del proyecto fue mejorar el aprendizaje del idioma inglés y, al mismo tiempo, adquirir experiencia en el trabajo colaborativo a distancia, fortaleciendo habilidades de comunicación intercultural y coordinación en entornos virtuales.
Proyecto 4

Creación de un sistema de gestión para un gimnasio utilizando PHP y SQL, integrando los conocimientos adquiridos en 1º DAW.

FitGestion
Este proyecto se desarrolló durante el primer curso del grado superior de Desarrollo de Aplicaciones Web (DAW) y consistió en la creación de un sistema de gestión para un gimnasio. Para su desarrollo se utilizaron tecnologías como PHP y bases de datos SQL. El objetivo del proyecto fue aplicar de forma práctica los conocimientos adquiridos en primero, integrando conceptos de programación, bases de datos y desarrollo web, con el fin de crear una solución funcional orientada a la gestión de usuarios, servicios y actividades del gimnasio.
